A web application that counts down to New Years and launches a spectacular confetti celebration when the clock strikes 12:00 AM

**Features**
Live Countdown Timer: Real-time countdown showing hours, minutes, and seconds until midnight
Automatic Confetti Celebration: Automatic fireworks display at exactly 12:00 AM
Multiple Confetti Effects:
        Continuous confetti bursts for 30 seconds
        Multi-directional fireworks
        Random color palettes and patterns
        Special celebration animations

**How to Use**
Simply open the index.html file in any modern web browser
The page will immediately show:
Current time
Countdown to New Years (in hours, minutes, seconds)
At midnight (12:00 AM), the celebration begins automatically:
      Celebration messages appear
      Confetti fireworks launch continuously for 30 seconds
      Special animations activate
